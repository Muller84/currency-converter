# Currency Converter with GUI

A lightweight Python application built with **tkinter** that fetches real-time exchange rates from an API to convert CZK to EUR and GBP.

## Features
- **Real-time Rates:** Uses [Exchangerate-API](https://www.exchangerate-api.com/) for up-to-date currency data.
- **Multi-currency:** Converts Czech Koruna (CZK) to Euro (EUR) and British Pound (GBP).
- **GUI:** User-friendly interface with error handling for invalid inputs.

## Requirements
- Python 3.x
- `requests` library

## How to Run
1. Install dependencies: `pip install requests`
2. Run the app: `python main.py`
